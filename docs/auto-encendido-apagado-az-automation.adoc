== Introducción

En este tutorial se indica cómo configurar el autoencendio y autoapagado de una máquina virtual en Azure. Esto es útil para evitar que la máquina virtual esté encendida y consuma recursos innecesarios, lo que puede resultar en costos adicionales.

Se describe como realizarlo con Logic Apps (consumption). Logic Apps es un servicio de Azure que permite automatizar flujos de trabajo y procesos empresariales.

== Cómo se factura Logic Apps 

La facturación de Logic Apps se basa en el consumo, lo que significa que solo pagas por las acciones y ejecuciones que realizas.

Actualmente Microsoft incluye por suscripción de forma gratuita un número limitado de ejecuciones de Logic Apps, que suele ser suficiente para la mayoría de los escenarios de automatización. Por ejemplo, si configuras una Logic App para encender y apagar una máquina virtual una vez al día, el número de ejecuciones será muy bajo y, por lo tanto, no incurrirás en costos adicionales significativos.

De hecho, el ahorro obtenido por apagar diariamente la VM suele ser muchísimo mayor que cualquier coste potencial de Logic Apps, por lo que es una solución muy rentable para gestionar el encendido y apagado automático de máquinas virtuales en Azure.
